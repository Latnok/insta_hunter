# Эталонные критерии блогеров для продавцов одежды

Использовать как короткий чеклист для поиска похожих Instagram-блогеров.

## Обязательная база
- профиль доступен
- есть reels
- есть живая речь в reels
- есть полезные transcript'ы, а не только музыка/шум
- контент регулярно выходит

## Сильные сигналы профиля
- fashion / style / outfit / look
- стиль / мода / женственность
- обзоры покупок
- находки
- распаковки
- WB / Wildberries
- UGC / creator
- сотрудничество / PR / brand work

## Сильные сигналы в reels
- примерка вещей
- обзор одежды
- подборка образов
- рекомендации, что купить
- сравнение вещей
- распаковка одежды
- сезонные образы
- базовый гардероб

## Сильные сигналы в речи
- называет категорию вещи: жакет, платье, юбка, футболка, костюм, обувь, сумка
- говорит про материал: трикотаж, ткань, качество
- говорит про посадку: как сидит, оверсайз, приталенное, длина
- говорит про цвет и сочетания
- говорит, кому подойдет вещь
- объясняет, с чем носить
- объясняет, почему вещь удобная / выгодная / универсальная
- упоминает, где купила / где заказать

## Коммерчески сильная подача
- понятная речь без мусора
- объяснение пользы вещи
- демонстрация вещи в образе
- демонстрация вещи в жизни, а не только крупный план
- естественная рекомендация товара
- есть ощущение, что блогер умеет продавать, а не просто снимать lifestyle

## Приоритетные типы блогеров
- fashion influencer
- style / outfit creator
- UGC creator с уклоном в одежду
- блогер с обзорами покупок и находок
- блогер с контентом про WB / маркетплейсы / подборки

## Слабые сигналы
- только beauty без одежды
- только lifestyle без товарного угла
- только музыка в reels
- нет внятной речи о вещах
- нет обзоров / примерок / подборок

## Мусорные transcript-сигналы
- DimaTorzok
- Субтитры сделал ...
- Субтитры создавал ...
- Субтитры подогнал ...
- Музыка

## Идеальный образец для поиска похожих
Блогер про стиль / находки / обзоры покупок / WB / UGC, у которого в reels есть речь про конкретные вещи: материал, посадку, цвет, сочетания, удобство, уместность и причина купить.
